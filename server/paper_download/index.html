文献属性

  • title：文献的名称

  • DOI：DOI的全称为“Digital Object Unique Identifier”, 中文意思即: 数字对象唯一标识符, 可以理解为数字资源唯一的”身份证号码“, 专门用来标记参考文献, 书籍资源和学术报告内容等等

  • URL：文献的网页链接，如NCBI链接等
